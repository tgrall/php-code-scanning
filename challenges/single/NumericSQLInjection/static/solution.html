Solution

The application is taking the input from the select box and inserting it at the end of a pre-formed SQL command. Compound SQL statements can be made by joining multiple tests with keywords like AND and OR. Try appending a SQL statement that always resolves to true.

This is the query: SELECT * FROM weather_data WHERE station = 1

Intercept the post request with a proxy and replace 1 with 1 or 1=1

As the SQL Statement is true for every station you get a list of all stations.

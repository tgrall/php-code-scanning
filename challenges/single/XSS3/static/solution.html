This lesson is using JavaScript hash based navigation. i.e the image that is displayed is based on the number after the hash in the url. View the source code of the page and look at the source of 'image.js' which contains the code for navigation.

Any value that you will supply in the url after the hash will be concatenated with the img tag and the new tag is placed inside the div. You can inject a DOM element of your own by crafting the url. For example : 3.jpg'/><img src=x onerror=alert(1)> will append another img tag by which an arbitary JavaScript code can be executed.
